Made for everyone

Whether you are a developer, a system administrator or just getting started with WinGet, WinGetty is the perfect solution for you.

Easy to use

With it's intuitive user interface you can add packages, add versions and installers and get started distributing your own packages within minutes.

Open Source

WinGetty is open source and free to use. The source code is available on GitHub and you can contribute to the project if you want to.

Fully private

WinGetty is fully private and does not require any Azure services allowing you to keep your packages and data on your own servers and under your control.

Easy as 1. 2. 3.

Just follow these three simple steps.

Install & Host

Just clone the repository or use the Docker image, configure the settings and host it on your own server.

Connect to Repository

Connect your Windows devices to your repository by running a single command. This is as easy as typing winget source add
