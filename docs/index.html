Powerful Features for Your Private Repository

WinGetty provides a comprehensive set of features to help you manage your private WinGet repository with ease.

Installer Switches

Easily manage installer switches for your packages, ensuring seamless installation.

Nested Installers

Support for nested installers, allowing you to manage complex package dependencies.

Multi-User Capabilities

Easy as 1 2 3

Just follow these three simple steps.

1. Set up WinGetty

Simply purchase one of the WinGetty Cloud plans and you will receive an instance within minutes. For advanced users, you can also deploy WinGetty on your own server.

2. Connect to Repository

Connect your Windows devices to your repository by running a single command. This is as easy as typing winget source add

3. Deploy

You are now ready to deploy your packages to your devices. Just run winget install <package> and WinGet will install your package.
